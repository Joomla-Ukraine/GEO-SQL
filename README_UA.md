# GEO-SQL

## Структура
* Країни
* Регіони
* Міста

**Країни:**
* Код країни
* Сортування
* Країна англійською
* Країна російською
* Країна українською

**Регіони:**
* Код країни
* ID регіону
* Регіон англійською
* Регіон російською
* Регіон українською (необхідна локалізація)

**Города:**
* Код країни
* ID регіону
* Позначення столиці (необхідно доповнити)
* Позначення мегаполіса, наприклад обласний центр (необхідна локалізація)
* Місто англійською
* Місто російською
* Місто українською (необхідна локалізація)
